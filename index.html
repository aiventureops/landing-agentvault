Everything your agents need to operate securely

🔐

Short-lived leases

Secrets are never long-lived tokens. Each access returns a time-bounded lease. A compromised agent can't drain your credentials.

🏗️

Per-agent namespaces

Each agent or workflow gets an isolated namespace. Agent A can never read Agent B's secrets — even in the same team.

📋

Full audit log

Every access, rotation, and deletion is logged with agent ID and timestamp. Know exactly which agent touched what and when.

🔌

MCP server included

Ships with a Model Context Protocol server. Claude, Cursor, and Windsurf agents can access secrets natively — no SDK required.

🔄

Auto-rotation

Set a TTL on any secret and it rotates automatically. Agents always get fresh credentials without any manual intervention.

🛡️

Zero plaintext in DB

AES-256-GCM encryption with envelope key management. A database breach exposes only ciphertext — useless without the master key.
